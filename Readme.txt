Urban Nest Solutions - Project Setup Guide
==========================================

Project Overview
----------------
Urban Nest Solutions is a modern e-commerce website for selling home and kitchen utility items such as:
- Fridge storage containers
- Masala boxes
- Cooking items
- Kitchen utensils
- Other home utility products

This project includes:
- index.html
- styles.css
- script.js
- assets folder for images and logo files

How to Use
----------
1. Download or create the project folder.
2. Place index.html, styles.css, and script.js in the main folder.
3. Add your images inside the assets/images folder.
4. Replace placeholder images with your real product photos and logo.
5. Open index.html in a web browser to view the website.

File Descriptions
------------------
index.html
- Main webpage structure
- Includes navigation, hero section, products, about section, contact form, and footer

styles.css
- Handles all design, colors, spacing, layout, and responsiveness

script.js
- Adds interactive features such as:
  - Smooth scrolling
  - Product search
  - Cart counter
  - Mobile menu toggle
  - Contact form behavior

Customization Notes
-------------------
- You can change the colors in styles.css to match your brand.
- You can update product names, prices, and descriptions in index.html.
- You can replace placeholder images with your own product images later.
- You can connect the contact form to email or backend service later.

Important Notes
---------------
- Keep image names and file paths exactly the same unless you update the HTML file too.
- Make sure all files are saved in UTF-8 encoding.
- If the website is hosted online, test it on mobile and desktop screens.
